## Overview

Warstellar empowers you to code faster and smarter with AI-driven automation. You collaborate seamlessly with intelligent agents that handle routine tasks, generate code, and optimize workflows. Built on advanced models, Warstellar transforms your development process from manual drudgery to efficient human-AI partnership.

Focus on creative problem-solving while Warstellar manages boilerplate, debugging, and refactoring. Whether you work solo or in teams, it integrates into your IDE and scales with your projects.

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Create a free account at https://dashboard.example.com/signup. No credit card required.
  </Step>
  <Step title="Install Extension" icon="download">
    Add the Warstellar extension to your IDE.

    <CodeGroup tabs="VS Code,JetBrains">
    ````bash
    code --install-extension warstellar.ai/warstellar
    ````
    ````bash
    # In JetBrains Toolbox or Settings > Plugins
    Search for "Warstellar" and install
    ````
    </CodeGroup>
  </Step>
  <Step title="First Prompt" icon="play">
    Open a file, type a comment like `// Generate a React component for user login`, and press Tab to accept AI suggestions.
  </Step>
</Steps>
